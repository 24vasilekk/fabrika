# Фабрика воспоминаний — современный многостраничный сайт

Редизайн в стилистике 2026: чистая типографика, карточки каталога с изображениями, CTA в Telegram, единый визуальный язык на всех страницах.

## Страницы

- `index.html` — главная: hero + интерактивный текст + каталог с карточками, фото и кнопками заказа.
- `services.html` — процесс работы.
- `portfolio.html` — сетка кейсов с быстрым способом подмены фото.
- `about.html` — о студии.
- `contact.html` — современные контакты и форма заявки.

## Ассеты

- `assets/logo.svg` — логотип.
- `assets/catalog/card-*.svg` — временные изображения для карточек каталога (можно заменить на реальные фото).
- `assets/portfolio/` — папка для ваших настоящих кейсов.

## Быстрый запуск

```bash
python3 -m http.server 8000
```

Откройте: `http://localhost:8000/index.html`.

## Как заменить фото в каталоге/портфолио

1. Положите файлы в `assets/portfolio/`.
2. В нужной карточке поменяйте `src` у `<img>` на свой файл.
